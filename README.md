🌌 3D Scroll Portfolio Website

A modern, interactive 3D scrolling website built using cutting-edge web technologies. This project combines smooth scroll animations with immersive 3D visuals to create an engaging user experience similar to Awwwards-style websites.

🚀 Features
🎮 Interactive 3D scene (WebGL based)
📜 Scroll-based animations (GSAP)
⚛️ Component-based architecture using HTML, CSS And JavaScript
🌐 Real-time 3D rendering using HTML, CSS And JavaScript
🧩 Simplified 3D integration with HTML, CSS And JavaScript
✨ Smooth UI animations with Framer Motion
📱 Fully responsive design
⚡ Optimized performance for modern browsers

🛠️ Tech Stack
Frontend: HTML, CSS And JavaScriptt
3D Graphics: HTML, CSS And JavaScript
Animations: GSAP, Framer Motion
Styling: CSS / Tailwind CSS
Version Control: Git & GitHub

⚙️ Installation & Setup
1. Clone the repository:
git clone https://github.com/your-username/3d-scroll-website.git
2. Navigate to the project folder:
cd 3d-scroll-website
3. Install dependencies:
npm install
Run the development server:
npm start
